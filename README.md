# Purpose

This project is a learning project designed to help me understand and implement AI workflows

# How this project was planned and built

I saw someone on LinkedIn mention building a project that search job board postings and having an AI agent compare the job descriptions to a resume and rank the roles returned. 

I decided this would be a good test project for me as well. I leveraged ChatGPT to help determine the requirements for such a project and had ChatGPT write the user stories to my JIRA instance using a Rovo plugin.

# Codex Usage
I had Codex read the requirements for a given story from Jira and write the code that will retrieve the job board postings, process and persist the results, and call the AI agent model.

# AI Workflow
I used ChatGPT and other resources to help me understand how to build out the workflow myself as this is the part I really wanted to learn.
